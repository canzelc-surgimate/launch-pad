What changes for users

Before

  • Legacy Home with chart wall — drill to Overview
  • Overview totals can disagree with source data (data integrity gap)
  • Analytics in a separate Superset session
  • No return path from dashboard insight to daily work
  • Dated nav chrome (pill controls, dense tabs)

After

  • Home workbench: MY TASKS, filterable table, customizable widgets
  • Governed KPIs via ClickHouse + Superset (schema-validated path)
  • Analytics Hub embedded in the DASHBOARD tab
  • Avoidable drill → View Detail → Home with preset filter
  • Modern flat nav, operational-first layout

ELT framing

Talking points

Data integrity

Overview drilldowns today can surface numbers that don’t reconcile with operational truth. The analytics pipeline + schema-validated KPIs are the fix — not another chart on Home.

Prototype scope

Home workbench uses representative mock data for UX validation. Executive Scorecard and Avoidable dashboards run on ClickHouse with real query logic.

Why it matters

Schedulers live in Surgimate daily. Executives live in analytics periodically. This design meets both in one product — insight without leaving the workflow.

What's next

Wire workbench to live APIs, expand widget catalog, harden embed auth, and prioritize KPIs from the Analytics Hub reference guide.
